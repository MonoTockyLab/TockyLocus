# Calculate Tocky Locus using Timer Angle

Calculate Tocky Locus using Timer Angle

## Usage

``` r
TockyLocus(x)
```

## Arguments

- x:

  A data.frame object or a TockyPrepData object

## Value

Input data frame object including Tocky Locus data.

## Examples

``` r
if (FALSE) { # \dontrun{
x <- TockyLocus(x)
} # }
```
