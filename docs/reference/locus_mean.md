# Calculate mean percentage for each locus using cellcount_table

Calculate mean percentage for each locus using cellcount_table

## Usage

``` r
locus_mean(x, sampledef = sampledef)
```

## Arguments

- x:

  A TockyPrepData or the output of cellcount_table

- sampledef:

  a sample_definition data frame

## Value

A summary table as a data frame.

## Examples

``` r
if (FALSE) { # \dontrun{
p <- locus_mean(x)
} # }
```
