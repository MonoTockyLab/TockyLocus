# Plot Coloured Tocky Locus Legend

Plot Coloured Tocky Locus Legend

## Usage

``` r
plotTockyLocusLegend(mar_par = c(4, 4, 10, 4))
```

## Arguments

- mar_par:

  parameters for the function mar. The default is c(4, 4, 10, 4)

## Value

A plot with colored rectangles and labels.

## Examples

``` r
if (FALSE) { # \dontrun{
  plotTockyLocusLegend()
} # }
```
