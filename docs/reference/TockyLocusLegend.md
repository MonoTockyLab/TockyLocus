# Generate Tocky Locus Legend For a Plot

Generate Tocky Locus Legend For a Plot

## Usage

``` r
TockyLocusLegend(legend = TRUE, cex = 1, viridis = FALSE)
```

## Arguments

- legend:

  A logical arguement.

- cex:

  A numeric value for the text size.

- viridis:

  (Optional). If TRUE, a colour-blind friendly colour set is used.

## Examples

``` r
if (FALSE) { # \dontrun{
TockyLocusLegend()
} # }
```
